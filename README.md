# Portafolio - Linwi Vargas Campos

## Sobre mí

Soy Realizadorx Audiovisual y Productor de videojuegos en formación, apasionadx por crear experiencias interactivas que visibilicen identidades y narrativas silenciadas. Mi objetivo es construir juegos que no solo entretengan, sino que también cuestionen perspectivas, fomenten la inclusión y abran espacios para mujeres, disidencias y comunidades diversas.

Actualmente me desempeño como Associate Producer y Productor en equipos internacionales, coordinando procesos, facilitando la comunicación entre áreas y asegurando que las ideas creativas se transformen en proyectos jugables y significativos. Encontré en los videojuegos un medio ideal para fusionar narrativa, estética y colaboración.

Paralelamente, estoy profundizando en el desarrollo web como Python Full Stack Developer, con especial interés en el Frontend. Me apasiona crear interfaces intuitivas y experiencias de usuario atractivas que combinen funcionalidad con diseño visual, aprovechando mi background audiovisual para construir productos web con identidad propia.

## Proyectos Destacados

### Game Production Manager
- **Descripción**: Sistema completo de gestión de producción de videojuegos que fusiona mi experiencia y problemáticas reales con desarrollo web
- **Demo en vivo**: [game-production-manager.onrender.com](https://game-production-manager.onrender.com/)
- **Repositorio**: [github.com/Linwi-V/GameProductionManager](https://github.com/Linwi-V/GameProductionManager)
- **Tecnologías**: Django, Python, PostgreSQL, Bootstrap, JavaScript, HTML5
- **Características**:
  - Gestión integral de proyectos de videojuegos
  - Sistema de seguimiento de tareas por áreas (programación, arte, diseño, audio)
  - Dashboard interactivo con métricas de progreso
  - Sistema de roles y permisos para equipos
  - Gestión de cronogramas y milestones
  - Interfaz diseñada específicamente para workflows de game development

### Portafolio Web Personal
- **Descripción**: Sitio web desarrollado con Django para mostrar mis proyectos de Desarrollo web y Videojuegos
- **Demo en vivo**: [portafolio-linwi.onrender.com](https://portafolio-linwi.onrender.com/)
- **Repositorio**: [github.com/Linwi-V/Portafolio](https://github.com/Linwi-V/Portafolio)
